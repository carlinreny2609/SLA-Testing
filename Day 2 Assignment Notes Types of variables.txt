5.Explain the different types of variables using an Example program?
----------------------------------------------------------------------

A variable is a element used to store and manipulate data throughout program execution.

Types are.
1.Local Variable
2.Instance variable
3.static variable -Remains static throughout the program
4.Parameters variable.

We can see them in detail below with example programs.

Local Variable:
----------------

This variable is declared inside the method,block, constructor.

Example Program: Here a,b,c are local variables.
================
package selenium;

public class Variabletypes {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		
		int a = 10;
		int b = 20;		
		int c= a+b;		
		System.out.print(c);

	}

}

instance variable:
------------------

An object is called as instance variable.


Example program:
-----------------

package selenium;

public class Variabletypes {
	
	  String Name = "Carlin";//Instance variable

	public static void main(String[] args) {
		// TODO Auto-generated method stub
				
		Variabletypes Variable = new Variabletypes();
		System.out.print(Variable.Name);
		
	}

}


Static Variable:
-----------------

This Variable is declared in a class but outside any method,block or constructor with a static keyword.


example Program:
----------------

package selenium;

public class Variabletypes {
	
	 static int age = 30;   //static Variable

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		Variabletypes Variable = new Variabletypes();
		System.out.print(Variable.age);//Instance variable
		
	}

}


Parameter Variables:
--------------------

In these variables the parameters are passed as arguments in the method,block,constructor to perform an action.

in the below example a, b are the parameters passed into the parameter variable method.

package selenium;

public class Variabletypes {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		parametervariable(10, 20);//integer parameter are passed in the arguments
	}

	public static void parametervariable(int a, int b)//integer parameters are passed in the arguments
	
	{
		int c = a + b;
		System.out.print(c);

	}
}
